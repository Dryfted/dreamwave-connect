# DreamWave Connect

Connect your Claude and Codex subscriptions to DreamWave for AI-powered terminal agents.

## Installation

### macOS / Linux
1. Download `dreamwave-connect-mac.tar.gz` or `dreamwave-connect-linux.tar.gz`
2. Extract: `tar -xzf dreamwave-connect-*.tar.gz`
3. Run: `./dreamwave-connect/dreamwave-connect`

### Windows
1. Download `dreamwave-connect-win.zip`
2. Extract the zip file
3. Run: `dreamwave-connect\dreamwave-connect.bat`

## Usage

1. The app will guide you through connecting your Claude and Codex accounts
2. Follow the terminal instructions to authenticate
3. Your credentials will be securely synced to DreamWave

## Requirements

- Node.js is not required - Electron is bundled
- On Linux, you may need to install: `sudo apt-get install libgtk-3-0 libnss3`
